## 2. products 产品主表

| 数据库字段 | 类型 | 约束 | 前端/API 字段名 | 用途说明 | 来源 |
|-----------|------|------|----------------|---------|------|
| `id` | `uuid` | PK, gen_random_uuid() | `id` | 产品唯一标识 | 系统生成 |
| `product_no` | `varchar(32)` | NOT NULL, UNIQUE | `productNo` | 产品货号/SPU编码，如 `YXGW260528-0001` | R2 同步 |
| `title` | `varchar(200)` | NOT NULL | `title` | 产品中文名称，如 "复古银色多层手链套装" | R2 同步 |
| `short_title` | `varchar(50)` | NULLABLE | `shortTitle` | 产品短标题 | R2 同步 |
| `category` | `varchar(20)` | NULLABLE | `category` | 产品类目，如 "手链"/"项链"/"耳环"/"戒指"/"发饰" | R2 同步 + 运营编辑 |
| `description` | `text` | NULLABLE | `description` | 产品详细描述（中文） | R2 同步 + 运营编辑 |
| `folder_name` | `varchar(300)` | NOT NULL | `folderName` | R2 中的素材包文件夹名，如 "[YGG00021] 银心格纹心形挂件_素材包" | R2 同步 |
| `r2_base_path` | `varchar(500)` | NULLABLE | `r2BasePath` | R2 存储中素材包的基础路径 | R2 同步（不可编辑） |
| `r2_base_url` | `varchar(500)` | NULLABLE | `r2BaseUrl` | R2 存储中素材包的基础URL | R2 同步（不可编辑） |
| `r2_synced_at` | `timestamp with tz` | NULLABLE | `r2SyncedAt` | 最近一次 R2 同步到此产品的时间 | R2 同步（不可编辑） |
| `pim_synced_at` | `timestamp with tz` | NULLABLE | `pimSyncedAt` | 最近一次 PIM 端同步处理的时间 | 同步服务写入 |
| `main_image_url` | `varchar(500)` | NULLABLE | `mainImageUrl` | 产品主图 URL（取 R2 images.main[0].url） | R2 同步 |
| `images_json` | `jsonb` | NULLABLE | `imagesJson` | R2 product.json 中的 images 对象 JSON | R2 同步 |
| `outer_packaging_json` | `jsonb` | NULLABLE | `outerPackagingJson` | 外箱包装信息 JSON | R2 同步 |
| `tool_version` | `varchar(20)` | NULLABLE | `toolVersion` | 分拣工具版本号 | R2 同步（不可编辑） |
| `status` | `varchar(16)` | NOT NULL, DEFAULT 'pending' | `status` | 产品状态: `pending`/`active`/`archived` | 系统 + 运营 |
| `created_at` | `timestamp with tz` | NOT NULL, DEFAULT now() | `createdAt` | 记录创建时间 | 系统自动 |
| `updated_at` | `timestamp with tz` | NOT NULL, DEFAULT now() | `updatedAt` | 记录最近更新时间 | 系统自动 |

### imagesJson 子结构 (ProductJsonR2Images)

| 字段 | 类型 | 说明 |
|------|------|------|
| `main` | `{fileName: string, url: string}[]` | 主图列表 |
| `sku` | `{fileName: string, url: string}[]` | SKU 图列表 |
| `detail` | `{fileName: string, url: string}[]` | 详情图列表 |

### outerPackagingJson 子结构 (OuterPackaging)

| 字段 | 类型 | 说明 |
|------|------|------|
| `length` | `number` | 外箱长 (cm) |
| `width` | `number` | 外箱宽 (cm) |
| `height` | `number` | 外箱高 (cm) |
| `weight` | `number` | 外箱毛重 (g) |
| `presetName` | `string` | 预设名称 |

## 3. product_skus SKU 规格表

| 数据库字段 | 类型 | 约束 | 前端/API 字段名 | 用途说明 | 来源 |
|-----------|------|------|----------------|---------|------|
| `id` | `uuid` | PK, gen_random_uuid() | `id` | SKU 唯一标识 | 系统生成 |
| `product_id` | `uuid` | NOT NULL, FK → products.id ON DELETE CASCADE | `productId` | 所属产品ID | 系统关联 |
| `sku_code` | `varchar(32)` | NOT NULL | `skuCode` | SKU 编码，如 "0001-S" | R2 同步 |
| `sku_name` | `varchar(100)` | NOT NULL | `skuName` | SKU 名称/规格名称，如颜色尺寸描述 | R2 同步 |
| `image_url` | `varchar(500)` | NULLABLE | `imageUrl` | SKU 图片 URL | R2 同步 |
| `original_image` | `varchar(200)` | NULLABLE | `originalImage` | 原始图片文件名 | R2 同步 |
| `size` | `varchar(100)` | NULLABLE | `size` | 尺寸规格，如 "16cm"/"2.5cm"/"可调节" | R2 同步 + 运营编辑 |
| `weight_g` | `integer` | NULLABLE | `weightG` | 单件重量 (克) | R2 同步 + 运营编辑 |
| `cost_price` | `numeric(10,2)` | NULLABLE | `costPrice` | 成本价 (元) | R2 同步 + 运营编辑 |
| `selling_price` | `numeric(10,2)` | NULLABLE | `sellingPrice` | 建议售价 (元) | R2 同步 + 运营编辑 |
| `stock` | `integer` | DEFAULT 0 | `stock` | 库存数量 | 运营编辑 |
| `barcode` | `varchar(50)` | NULLABLE | `barcode` | 条形码 | 运营编辑 |
| `sort_order` | `integer` | DEFAULT 0 | `sortOrder` | 排序顺序 | R2 同步 |
| `created_at` | `timestamp with tz` | NOT NULL, DEFAULT now() | `createdAt` | 创建时间 | 系统自动 |
| `updated_at` | `timestamp with tz` | NOT NULL, DEFAULT now() | `updatedAt` | 更新时间 | 系统自动 |

### 同步字段保护策略

同步不会覆盖运营已填写的以下字段（仅当原有值为 NULL 时才填入）：

| 保护字段 | 说明 |
|---------|------|
| `size` | 尺寸 |
| `weight_g` | 重量 |
| `cost_price` | 成本价 |
| `selling_price` | 售价 |
| `stock` | 库存 |
| `barcode` | 条形码 |

- **不可编辑字段**: `productNo`, `r2BasePath`, `r2BaseUrl`, `r2SyncedAt`, `toolVersion`, `createdAt`, `id`
- **运营可编辑字段**: `title`, `shortTitle`, `category`, `description`, `folderName`, `pimSyncedAt`, `mainImageUrl`, `imagesJson`, `outerPackagingJson`, `status`

## 7. distributors 分销商表

| 数据库字段 | 类型 | 约束 | API 字段名 | 用途说明 |
|-----------|------|------|-----------|---------|
| `id` | `uuid` | PK | `id` | 分销商ID |
| `name` | `varchar(100)` | NOT NULL | `name` | 分销商名称 |
| `contact` | `varchar(100)` | NULLABLE | `contact` | 联系方式 |
| `api_token` | `varchar(100)` | NOT NULL, UNIQUE | `apiToken` | API Token（64位hex） |
| `allowed_categories` | `text[]` | NULLABLE | `allowedCategories` | 允许访问的类目列表，NULL=全部 |
| `price_type` | `varchar(20)` | DEFAULT 'selling' | `priceType` | 定价方式: `selling`/`cost`/`custom` |
| `price_markup` | `numeric(5,2)` | DEFAULT '0' | `priceMarkup` | 加价百分比 (仅 custom 模式生效) |
| `is_active` | `boolean` | DEFAULT true | `isActive` | 是否启用 |
| `created_at` | `timestamp with tz` | NOT NULL | `createdAt` | 创建时间 |

### 分销商价格计算规则

| priceType | 计算逻辑 | 返回 |
|-----------|---------|------|
| `selling` | 直接返回 `selling_price` | `number \| null` |
| `cost` | 直接返回 `cost_price` | `number \| null` |
| `custom` | `cost_price × (1 + price_markup / 100)` | `number \| null` |
